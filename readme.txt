Состав релиза
Файлы:

vm.h — заголовочный файл VM

vm.c — виртуальная машина

compiler.c — компилятор

main.c — точка входа

Ключевые характеристики:

Единый числовой тип V_NUMBER (64.32 fixed point)

7 типов данных: V_NUMBER, V_STRING, V_ARRAY, V_UNDEF, V_HASH, V_FLOAT (-e), V_EDECIMAL (-e)

58 опкодов занято, ~35 свободны

Пробел как разделитель аргументов по умолчанию

Метки циклов last/next/redo

Именованные константы с подстановкой на этапе компиляции

Каскадная компиляция модулей .osm → .ocm

Режим -s для source comments

Режим -e для extended syntax
