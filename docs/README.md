# Oyster Documentation

## Language Reference

### Variables
- `$integer` - Integer (64-bit)
- `%decimal` - Decimal (double)
- `@array` - Array
- `@string` - String
- `&hash` - Hash (via stdlib)

### Control Flow
- `if/else`
- `while`
- `for in`

### Functions
fun name($param) {
    return $param
}

## Standard Library
- `io` - Input/Output operations
- `net` - Networking
- `regex` - Regular expressions
- `math` - Mathematical functions
